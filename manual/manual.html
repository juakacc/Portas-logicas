Manual de "Logic-man, o jogo"

Abaixo é exibida a tela inicial do jogo

Nessa tela o jogador poderá escolher entre as opções:

  • Iniciar jogo
  • Opções
  • Regras

Iniciar jogo: Clicando nesse botão o jogador será direcionado para a fase inicial do jogo.

FOTO INICIAR JOGO

Opções: Clicando nesse botão será mostrado para o jogador uma lista de opções, dentre elas os níveis do jogo, em que ele pode escolher ir diretamente para alguma fase.

FOTO DO MENU OPÇÔES

Regras: Clicando nesse botão será mostrado para o jogador a lista de regras do jogo, é recomendado a leitura com atenção para realizar um bom jogo.

Para oculta-la basta o jogador clicar no retângulo das regras

FOTO DAS REGRAS

Jogando...

Ao iniciar o jogo o jogador é redirecionado para o ambiente onde se passa o roteiro do jogo, naturalmente ele se inicia no nível 1, sua interface é mostrada na figura abaixo:

FOTO DO NIVEL 1

A interface do jogo é acompanhada de diversas funcionalidades, são elas:

  1. Painel de informações com: pontuação atual, número de vidas e nível atual
  2. Local para recolhimento dos conectivos lógicos
  3. Botão de menu
  4. Controle de som
  5. Botão de pausa
  6. Botão de ajuda
  7. Jogador
  8. Portas
  9. Ambiente de movimentação

O jogador poderá percorrer livremente os corredores da escola, recolhendo os símbolos e abrindo as portas

Ao recolher todos os conectivos lógicos, eles ficarão a disposição na parte superior, designada para eles

Botão de ajuda

Ao clicar no botão de ajuda será exibido para o jogador a tela de ajuda, nela estão dispostas algumas funcionalidades:

FOTO DA TELA DE AJUDA
  1. Menu para ajuda sobre os conectivos lógicos
  2. Botão para regras
  3. Botão fechar

Ao abrir todas as portas de um nível será exibido um cano, de cor verde, o jogador pode utilizá-lo para passar de nível

FOTO DO CANO

Ao passar de nível ele será redirecionado para o nível seguindo, seguindo as mesmas regras, mas agora em um ambiente diferente do anterior.

FOTO DO NÌVEL 2 COM MONSTRO

No nível 2 a dificuldade aumenta, tanto na dificuldade das questões, como também para percorrer os corredores, agora temos um monstrinho na sua perseguição.
